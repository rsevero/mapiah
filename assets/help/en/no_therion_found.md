# No Therion found

Mapiah needs access to a previously installed Therion to work. If you see this message, it means that Mapiah cannot find a Therion installation on your system.

If you don´t have Therion installed, you can download it from the [Therion download page](https://therion.speleo.sk/download.php).

If you already have Therion installed, make sure that it is working properly.

If still Mapiah can't find Therion, you can specify the path to the Therion executable in Mapiah settings.
