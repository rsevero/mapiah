# Therion não encontrado

O Mapiah precisa de acesso a uma instalação prévia do Therion para funcionar. Se você está vendo esta mensagem, significa que o Mapiah não conseguiu localizar uma instalação do Therion no seu sistema.

Se você não tem o Therion instalado, pode baixá-lo na [página de downloads do Therion](https://therion.speleo.sk/download.php).

Se você já tem o Therion instalado, verifique se ele está funcionando corretamente.

Se mesmo assim o Mapiah não conseguir encontrar o Therion, você pode especificar o caminho para o executável do Therion nas configurações do Mapiah.
