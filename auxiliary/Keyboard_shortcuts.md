| Add line segment between selected line segments | /                            |
| Add image                                       | Ctrl+Alt+I                   |
| Change active scrap                             | Alt+C                        |
| Change images                                   | Alt+I                        |
| Create area                                     | A                            |
| Create line                                     | L                            |
| Create point                                    | P                            |

| Open file                                       | Ctrl+O                       |
| Options edit                                    | O                            |
| Line node edit                                  | N                            |
| Redo                                            | Ctrl+Y                       |
| Save file                                       | Ctrl+S                       |
| Save file as                                    | Shift+Ctrl+S                 |
| Select all                                      | Ctrl+A                       |
| Select none                                     | Esc                          |
| Select tool                                     | C                            |
| Undo                                            | Ctrl+Z                       |
| Zoom 1:1                                        | 1                            |
| Zoom file                                       | 3                            |
| Zoom in                                         | +                            |
| Zoom out                                        | -                            |
| Zoom scrap                                      | 4                            |
| Zoom selection                                  | 2                            |
